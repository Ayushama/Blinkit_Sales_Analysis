#  Blinkit Sales Analysis Dashboard

##  Project Overview
This project analyzes **Blinkit sales data** to uncover key business insights related to product performance, outlet efficiency, customer preferences, and sales trends. The analysis combines **SQL for data cleaning & querying** and **Power BI for interactive visualization**, making it an end-to-end data analytics project.

---

## Objectives
- Analyze overall sales performance and key KPIs  
- Understand sales distribution across item types and fat content  
- Evaluate outlet performance by size, type, location, and establishment year  
- Identify high-sales but low-rated product categories  
- Build an interactive Power BI dashboard for business decision-making  

---

## Dataset Description
The dataset contains transactional-level sales data with the following attributes:
- Item details (type, fat content, visibility, weight)
- Outlet details (type, size, location, establishment year)
- Sales amount and customer rating  

---

## Data Cleaning & Preparation (SQL)
- Renamed columns using **snake_case** for consistency  
- Standardized categorical values (e.g., *Low Fat*, *Regular*)  
- Handled data types for numerical analysis  
- Aggregated data to generate KPIs and business metrics  

---

## Tools & Technologies
- **SQL (MySQL)** – Data cleaning, transformation, and analysis  
- **Power BI** – Data modeling, DAX measures, and dashboard creation  
- **Git & GitHub** – Version control and project hosting  

---

## Key KPIs
- **Total Sales**
- **Average Sales**
- **Total Transactions**
- **Average Rating**

---

## Dashboard Insights
The Power BI dashboard provides:
- Sales by Item Type (Top-performing categories)
- Sales by Outlet Type, Size, and Location
- Sales trend by Outlet Establishment Year
- Fat Content analysis (Low Fat vs Regular)
- Outlet age group performance
- Identification of high-sales but low-rated items
